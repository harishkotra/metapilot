# Types Directory

This directory contains TypeScript type definitions for MetaPilot.

## Structure

- `wallet.ts` - Wallet and account types
- `permissions.ts` - ERC-7715 permission types
- `agent.ts` - AI agent and execution types
- `envio.ts` - GraphQL response types

## Guidelines

- Comprehensive type coverage
- Clear naming conventions
- JSDoc comments for complex types
- Export types from index.ts
