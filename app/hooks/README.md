# Hooks Directory

This directory contains custom React hooks for MetaPilot.

## Structure

- `useWallet.ts` - Wallet connection and state management
- `usePermissions.ts` - Permission management hooks
- `useAgent.ts` - Agent execution hooks
- `useEnvio.ts` - GraphQL data fetching hooks

## Guidelines

- Follow React hooks conventions
- Handle loading and error states
- Provide TypeScript types for all hooks
- Include JSDoc documentation
